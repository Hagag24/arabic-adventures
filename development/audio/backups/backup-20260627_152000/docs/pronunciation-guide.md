## Vocalization Guidelines
- **Modern Standard Arabic (MSA)**: All core questions, instruction text, cards, and facts must use proper MSA rules.
- ** Egyptian Accent Narration**: MSA must be read with clear, calm Egyptian articulation (avoiding theatrical performances or non-Egyptian dialects).
- **Feedback Exception**: Short reinforcing notifications use friendly local Egyptian Arabic dialect ("برافو عليك، إجابتك صح!").

# Development Directory

This directory contains development-only tooling, documentation, reports, state files, quarantine folders, and backups for the Arabic Adventures project.

## Structure

- `audio/`
  - `tools/`: CLI tools for generation, auditing, and verification.
  - `docs/`: Audio generation guides, progress reports, etc.
  - `reports/`: Audit and cleanup reports.
  - `state/`: Persistent state of the audio generation process.
  - `quarantine/`: Quarantined invalid or fake placeholder files.
  - `backups/`: Timestamped backups of project states.
- `tests/`: Development-only test suites and fixtures.
